# Nova-Mart

**Nova-Mart** is a modern, responsive furniture e-commerce website built with React and TypeScript. It allows users to browse, search, and purchase furniture items with a seamless shopping experience.

---

## Features

- Responsive UI optimized for desktop and mobile devices  
- Product listing with images, descriptions, and prices  
- Search and filter functionality  
- Shopping cart with add/remove items  
- User authentication (optional, can be added)  
- Smooth navigation and UI interactions  

---

## Tech Stack

- TypeScript  
- CSS / Tailwind / Styled-Components (specify your styling choice)
